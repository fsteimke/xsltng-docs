7.2. Codelisten-Regeln

All jene Regeln, die Aussagen über die Verwendung von Codelisten im Katalog treffen.

Identifier Übersetzung Fehlerart Informationselement
PEPPOL-T01-CL001

Der Allowance Reason code ("Document level allowance or charge reason code" (BT-T01-109), Document level allowance or charge reason code) muss einem Code aus der Teilmenge der Codeliste UNCL 5189 in der Version D.16B entsprechen.

 fatal BT-T01-109
T01-CL001

Der Wert von "Document level allowance or charge reason code" (BT-T01-109) muss Teil der Codeliste UNTDID 5189 Allowance or charge identification code sein, wenn der Wert von "Allowance Charge Indicator" (BT-T01-108) "falsch" entspricht.

 BT-T01-108 BT-T01-109
PEPPOL-T01-CL002

Der Charge Reason code ("Document level allowance or charge reason code" (BT-T01-109), Document level allowance or charge reason code) muss einem Code aus der Codeliste UNCL 7161 in der Version D.16B entsprechen

 fatal BT-T01-109
T01-CL002

Der Wert von "Document level allowance or charge reason code" (BT-T01-109) muss Teil der Codeliste UNTDID 7161 Special service description code sein, wenn der Wert von "Allowance Charge Indicator" (BT-T01-108) "true" entspricht.

 BT-T01-108 BT-T01-109
T01-CL003

Der Wert von "Line level allowance or charge reason code" (BT-T01-137) muss Teil der Codeliste UNTDID 5189 Allowance or charge identification code sein, wenn der Wert von "Charge Indicator" (BT-T01-136) "falsch" entspricht.
