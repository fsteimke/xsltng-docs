3.4. Kerndatenobjekt Eingetragener Name

3.4.1. Element ename:eingetragenerName

[Option 3/3] Datentyp: bdt:String.DIN91379.C (2.1.1 )

Die im Handels-, Genossenschafts- oder Gesellschaftsregister eingetragene Firma eines wirtschaftlich Tätigen oder einer wirtschaftlichen Tätigkeit (z. B. eingetragenes Einzelunternehmen, eingetragene Zweigniederlassung) bzw. der im Vereins-, Partnerschafts- oder Stiftungsregister oder im Stiftungsverzeichnis eingetragene Name.

Der eingetragene Name kann auch in einem Register festgehalten sein, welches in einem anderen Staat geführt wird.

Der eingetragene Name ist abzugrenzen von der Geschäftsbezeichnung.

Dieses Element ist eine Ausprägung des Kerndatenobjekts

urn:xoev-de:xunternehmen:kerndatenobjekt:eingetragenername

[Option 3/3] aus dem Standard XUnternehmen.Kerndatenmodell.
