5.2. Kerndatenobjekt Name einer natürlichen Person

5.2.1. Element np:name

Datenobjekt zur Darstellung des Namens einer natürlichen Person in lateinischer Schrift, entsprechend der Vorgaben aus dem Baukasten des Föderalen Informationsmanagements (FIM).

Dieses Element ist eine Ausprägung des Kerndatenobjekts

urn:xoev-de:xunternehmen:kerndatenobjekt:namenatuerlicheperson

[Option 3/3] aus dem Standard XUnternehmen.Kerndatenmodell.

Kindelemente von name
Kindelement Typ Anz. Ref. Seite
np:familienname (ref) bdt:String.DIN91379.C 0..1 5.2.4

Familienname einer natürlichen Person bestehend aus Nachname, Zuname bzw. Familienname.
np:familienname.nichtVorhanden (ref) xs:boolean 0..1

Angabe, dass die betreffende Person keinen Familiennamen führt.

Dieses Element ist nur anzugeben, wenn es den Wert true hat.
np:geburtsname (ref) bdt:String.DIN91379.C 0..1 5.2.6

Der Geburtsname entspricht dem Familiennamen einer natürlichen Person vor der Schließung der ersten Ehe bzw. vor dem Eingehen der ersten Lebenspartnerschaft.
np:geburtsname.nichtVorhanden (ref) xs:boolean 0..1

Angabe, dass vor der Eheschließung oder vor der Begründung der Lebenspartnerschaft nach dem für die Namensführung maßgebenden Recht kein Familienname vorhanden gewesen ist.

Dieses Element ist nur anzugeben, wenn es den Wert true hat.
np:vornamen (ref) bdt:String.DIN91379.C 0..1 5.2.2

Vornamen einer natürlichen Person.
np:vornamen.nichtVorhanden (ref) xs:boolean 0..1

Angabe, dass die betreffende Person keinen Vornamen führt.

Dieses Element ist nur anzugeben, wenn es den Wert true hat.
